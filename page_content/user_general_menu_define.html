Define menu



This menu allows you to define the architecture of your project source code in order to check the compliance against the actual architecture. For more specific information on this component, we hereby refer you to the user documentation of define.

Define architecture

This enables the user to define a logical architecture and map an application to it. It is also possible to set architectural rules.
For more details on this component refer to the Define user manual.

Define architecture diagram

This shows the user a graphical representation of the defined architecture.
For more details on this component refer to the Graphics user manual.

Import architecture

Import a previously saved architecture into HUSACCT. The architecture file is XML.

Export architecture

Export a defined architecture to file. HUSACCT only supports XML for exporting the architecture.
