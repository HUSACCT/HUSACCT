Download and run HUSACCT_3.x

1.       Visit: http://husacct.github.io/HUSACCT/

At this site you can watch an introduction video, access the documentation and download the latest release of HUSACCT.
Select “Download HUSACCT_x.x JAR File” and save the jar in a suitable directory.

2.       Open/Start the HUSACCT jar

a.       The tool starts when you open or double click the jar-file (if your local Java-settings are ok).

b.      If not (and you are running Windows), create a shortcut and edit Target to:
java -jar <pathToHUSACCT>HUSACCT_3.x.jar
For example:
"C:\Program Files (x86)\Java\jre7\bin\java.exe" -jar C:\Tools\HUSACCT\HUSACCT_3.2.jar

c.       Alternatively, create a .bat file to start HUSACCT. For example, to start HUSACT_3.2:

                           i.       create a txt file in the same dir as HUSACCT and type: java -jar HUSACCT_3.2.jar

                         ii.      Save this file as: HUSACCT_3.2Start.bat

                        iii.      Double click this .bat file will start up HUSACCT.

d.      Alternatively, start a Command prompt, cd to the directory with the HUSACCT jar and start the tool from the command line, for example: C:\Tools>java -jar HUSACCT_3.2.jar.
Note: HUSACCT 3.x requires Java 1.7. If needed, check the Java version: java –version.
