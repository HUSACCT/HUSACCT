Download and Run

1. Select: Download HUSACCT_x.x JAR File.
Save the jar in a suitable directory.

2. Open/Start the HUSACCT jar
a. The tool starts when you open or double click the jar-file (if your local Java-settings are ok).

b. If not, create a shortcut and edit Target e.g. to (in case of Windows):
java -jar HUSACCT_5.1.jar
For example: "C:\Program Files\Java\jre1.8.0_73\bin\java.exe" -jar C:\Tools\HUSACCT\HUSACCT_5.1.jar

c. Alternatively, create a .bat file to start HUSACCT. For example, to start HUSACT_5.1:
Create a txt file in the same dir as HUSACCT and type: java -jar HUSACCT_5.1.jar
Save this file as: HUSACCT_5.1Start.bat
Double clicking on this .bat file will start up HUSACCT.

d. Alternatively, start a Command prompt.
Go to the directory with the HUSACCT jar and start the tool from the command line.
For example: C:\Tools>java -jar HUSACCT_5.1.jar.

Note 1: HUSACCT 5.5 is developed with Java 11. If needed, download a recent JRE to run HUSACCT.
Note 2: In case of large systems, you can get an OutOfMemoryError with the message: Java heap space. With the -Xmx JVM argument, you can set the heap size. For instance, for systems over 1.000.000 lines of code, allow the JVM to use 4 GB of memory. E.g. with the following command in a (Windows) shortcut or bat file:
java -jar -Xmx4g HUSACCT_5.1.jar
