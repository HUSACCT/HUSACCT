Main HOI GUI

Menu Bar

#1 Zoom in

Click to zoom in.

#2 Zoom out

Click to zoom out.

#3 Refresh

Refresh the diagram.

#4 Export to image

Click to export the current diagram to an image file.

#5 Hide selected modules

Click to hide the modules that have been selected using the select tool.

#6 Restore hidden modules

Restore previously hidden modules to diagram.

#7 Show dependencies

A checkbox that reflects whether or not dependency arrows are currently displayed.

#8 Show external libraries

A checkbox that reflects whether or not external libraries are currently displayed.

#9 Show violations

A checkbox that reflects whether or not violation arrows are currently shown.

#10 Smart lines

A checkbox that displays whether or not "smart" lines are currently enabled. When enabled, thicker lines mean higher numbers.

#11 Select layout strategy

Select the layout strategy to use when displaying new diagrams.

#12 Zoom slider

Slide to zoom.
