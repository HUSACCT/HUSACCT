Export Dependencies

This area gives the possibility to export a list of all identified dependencies in a .xls file.
To export a list of all dependencies you will need to do the following steps:

1. Select “Export Dependencies”.
2. Click on the “Browse” button in the Export Dependencies section.
3. Browse a location to export the list. 4. Click on the “Export” button.

After a while the file will be exported on the location you selected.
